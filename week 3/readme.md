SOC Analyst — Week Assignment

Overview
This repository contains both theoretical understanding and hands-on practical implementations completed as part of SOC Analyst training. The focus areas include log analysis, threat intelligence integration, incident response workflows, alert triage, evidence handling, and a complete SOC workflow simulation.

Key Learning Areas

- **Advanced Log Analysis**
  - Log correlation and anomaly detection  
  - Data enrichment using GeoIP  

- **Threat Intelligence Integration**
  - IOC matching using threat feeds  
  - Alert enrichment with external intelligence  

- **Incident Escalation Workflows**
  - Case management and escalation procedures  
  - SITREP documentation  

- **Alert Triage**
  - Prioritization and validation using threat intelligence  

- **Evidence Preservation**
  - Volatile data collection  
  - Chain-of-custody and forensic integrity  

- **Capstone SOC Simulation**
  - End-to-end attack detection, response, and reporting  

Objectives

- Develop practical SOC investigation skills  
- Understand real-world attack detection techniques  
- Apply threat intelligence to enhance alert accuracy  
- Practice structured incident response workflows  
- Gain hands-on experience with industry tools
